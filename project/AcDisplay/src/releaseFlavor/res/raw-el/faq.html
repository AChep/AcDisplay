Πώς μπορώ να απεγκαταστήσω αυτή την εφαρμογή;
Πηγαίντε στις Ρυθμίσεις > Ασφάλεια > Διαχειριστές συσκευής και απεπιλέξτε την AcDisplay. Μπορείτε τώρα να απεγκαταστήσετε το AcDisplay είναι κανονικά.

Πώς απενεργοποιω τοίς σταθερές ενημερώσεις του active mode ή τής οθόνης κλειδώματος;
Αυτή η ειδοποίηση είναι απαραίτητη για να εξασφαλίσουμε ότι η AcDisplay δεν θα απενεργοποιηθεί από το σύστημα. Ωστόσο, για να την αποκρύψετε ασφαλώς (η δοκιμαστική ειδοποίηση δεν θα λειτουργεί πλέον!) πηγαίντε στις πληροφορίες εφαρμογής και απενεργοποιήστε την εμφάνιση ειδοποιήσεων.

Βλέπω ένα τυπογραφικό λάθος στη μετάφραση!
Ευχαριστούμε, μπορείτε εύκολα να βοηθήσετε με αυτό. Εγγραφείτε στην σελίδα της εφαρμογής στο crowdin και προτείνετε μια καλύτερη μετάφραση.

Πώς να χρησιμοποιήσω την μαύρη λίστα;
Για να χρησιμοποιήσετε τη μαύρη λίστα, πρέπει να ενεργοποιήσετε την εφαρμογή και καθώς και να την ρυθμίσετε. Για να την ενεργοποιήσετε, απλά κάντε κλικ στο κουμπί-διακόπτη. Για να ρυθμίσετε την εφαρμογή, κάντε κλικ στο στοιχείο που θέλετε να ρυθμίσετε.
